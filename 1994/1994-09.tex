{\bf 1. feladat: } Igazoljuk, hogy a $2p + 1$ alakú számok között, ahol $p$ prímszám, pontosan egy olyan van, amely egy pozitív egész szám köbe!


\ki{Szabó Magda}{Szabadka}\medskip

{\bf 2. feladat: } Hány olyan --- legalább kételemû --- halmaz van, amelynek elemei egymást követõ pozitív egész számok, és a halmaz elemeinek összege 100?


\ki{Kántor Sándorné}{Debrecen}\medskip

{\bf 3. feladat: } Egy kör $AB$ átmérõjét messük el egy, az $AB$-vel $45^{\circ}$-os szöget bezáró $CD$ húrral, $AB$ és $CD$ metszéspontját jelölje $P$! Igazoljuk, hogy $2(CP^2 + PD^2) = AB^2$.


\ki{Benedek Ilona}{Vác}\medskip

{\bf 4. feladat: } Határozzuk meg a $2x^2 - 8xy + 17y^2 - 16x - 4y + 2062$ kifejezés legkisebb értékét, ha $x$ és $y$ tetszõleges valós számok!


\ki{Róka Sándor}{Nyíregyháza}\medskip

{\bf 5. feladat: } Egy 2 egység kerületû háromszög oldalainak hossza $a$, $b$ és $c$. Igazoljuk, hogy
$$\begin{array}{lll}
&&a) \ (1-a)(1-b)(1-c) > 0 \\[6pt]
&&b) \ a^2 + b^2 + c^2 + 2abc < 2.
\end{array}$$


\ki{Szabó Magda}{Szabadka}\medskip

{\bf 6. feladat: } Bizonyítsuk be, hogy ha $x \in \mathbb{R}$ és $n>0$, akkor
$$\left[ {(n+1)x \over 2} - {[x] + [2x] + \ldots + [nx] \over n} \right] = 0,$$
ahol $[A]$ jelöli az $A$ szám egész részét, tehát azt a legnagyobb egész számot, amely nem nagyobb $A$-nál.


\ki{Bencze Mihály}{Brassó}\medskip
