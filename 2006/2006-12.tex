{\bf 1. feladat: } Legyen $x_1,x_2,\dots,x_n$  egész számok olyan sorozata, amelyre fennáll, hogy $-1\leq x_i\leq 2$, ha $i=1,2,\dots,n$, $x_1+x_2+\dots+x_n = 19$  és $x_1^2+x_2^2+\dots x_n^2 = 99$. Legyen az $x_1^3+x_2^3+\dots x_n^3$  kifejezés lehetséges legkisebb értéke $m$, illetve lehetséges legnagyobb értéke $M$.  Mivel egyenlõ $\frac{M}{m}$  értéke? 



\ki{Kiss Sándor}{Nyíregyháza}\medskip

{\bf 2. feladat: } Az adott konvex $\alpha$ szög belsõ tartományában van a $P$ pont. Hogyan kell a $P$ ponton keresztül meghúzni egy olyan egyenest, hogy a szög szárait $B$ és $C$ pontban messe, és az $\frac{1}{|BP|} + \frac{1}{|PC|}$ a lehetõ legnagyobb legyen? 




\ki{Szabó Magda}{Szabadka}\medskip

{\bf 3. feladat: } Hozzuk a legegyszerûbb alakra a $3+2\cdot 3^2 + 3\cdot 3^3 + \dots + 2005\cdot 3^{2005} + 2006\cdot 3^{2006}$ összeget. 





\ki{Oláh György}{Komárom}\medskip

{\bf 4. feladat: } Az $ABCD$ paralelogrammában a $BD$ átló $12$, az átlók metszéspontja $O$. Az $AOD$ és $COD$ háromszögek köré írható körök középpontjainak a távolsága $16$. Az $AOB$ háromszög köré írt kör sugara $5$. Határozzuk meg a paralelogramma területét.   



\ki{Pintér Ferenc}{Nagykanizsa}\medskip

{\bf 5. feladat: } Bizonyítsuk be, hogy nincs olyan $P(x)$  egész együtthatós (nem azonosan nulla) polinom, amelyre léteznek $x_1,x_2,\dots,x_n$ ($n>2$) különbözõ egész számok úgy, hogy $P(x_1) = x_2, P(x_2) = x_3, \dots, P(x_n) = x_1$ teljesül.   





\ki{Kántor Sándor}{Debrecen}\medskip

{\bf 6. feladat: } Egy $10\times 10$-es négyzetbe beírjuk az $1, 2, 3,\dots, 100$ számokat úgy, hogy bármely két egymás utáni szám élben szomszédos négyzetbe kerüljön. Igazoljuk, hogy létezik legalább egy sor vagy oszlop, amelyik legalább két teljes négyzetet tartalmaz. 




\ki{Bence Mihály}{Brassó}\medskip
