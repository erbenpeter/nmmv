{\bf 1. feladat: } Igazoljuk, hogy ha $x,y,z \in \mathbb{R}$, akkor
$$x^2 + y^2 + z^2 - xy - yz - zx \geq {3 \over 4} \max \left[ (x-y)^2, (y-z)^2, (z-x)^2 \right].$$


\ki{Bencze Mihály}{Brassó}\medskip

{\bf 2. feladat: } Hány olyan háromszög van, amelynek oldalai $n$-nél nagyobb, de $2n$-nél nem nagyobb egész számok? Ezek közül a háromszögek közül hány egyenlõszárú és hány egyenlõoldalú van?


\ki{Urbán János}{Budapest}\medskip

{\bf 3. feladat: } Jelölje $N$ azt az 1992 jegyû számot, amelynek az összes számjegye 9-es. Mennyi $N^2$ számjegyeinek összege?


\ki{Bencze Mihály}{Brassó}\medskip

{\bf 4. feladat: } Bizonyítsuk be, hogy
$$82!\left( 1 + {1 \over 2} + {1 \over 3} + \ldots + {1 \over 82} \right)$$
osztható 1992-vel.


\ki{Mészáros József}{Galánta}\medskip

{\bf 5. feladat: } Adott $ABC$ háromszög. Legyen $O$ a körülírt körének a középpontja. $B$ és $C$ csúcsokból az $AC$ és $AB$ oldalakra bocsátott merõlegesek talppontjai $E$ és $F$. Igazoljuk, hogy $AO \perp EF$.


\ki{Nagel tétele}{}\medskip

{\bf 6. feladat: } Az $ABC$ derékszögû háromszög $S$ súlypontjából bocsássunk merõlegeseket az oldalakra. Legyenek ezek talppontjai $A_1, B_1, C_1$. Számítsuk ki a Ter($ABC$)/Ter($A_1B_1C_1$) arányt.


\ki{Mészáros József}{Galánta}\medskip
