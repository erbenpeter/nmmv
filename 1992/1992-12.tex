{\bf 1. feladat: } Határozzuk meg azon $x,y$ egészeket, amelyekre $x^2(x^2 + 4xy + 3y^2)$ és $y^2(y^2 + 4xy + 3x^2)$ kifejezések egyszerre teljes negyedik hatványok.


\ki{Bencze Mihály}{Brassó}\medskip

{\bf 2. feladat: } Egy konvex 10-szög belsejében vegyünk fel $k$ pontot úgy, hogy bármely két pont összekötõ egyenese ne tartalmazzon sem a felvett pontok, sem a sokszögcsúcsok közül még egyet. Bontsuk fel a sokszöget háromszögekre úgy, hogy minden háromszög csúcsa csak a sokszögcsúcsokkal vagy pedig a felvett pontokkal esik egybe. Bizonyítsuk be, hogy bármilyen módon bontjuk fel a sokszöget háromszögekre, a háromszögek száma mindig ugyanakkora.


\ki{Reiman István}{Budapest}\medskip

{\bf 3. feladat: } Igazoljuk, hogy
$${1 \over 3!} + {1 \over 5!} + {1 \over 7!} + \ldots$$
irracionális.


\ki{Bencze Mihály}{Brassó}\medskip

{\bf 4. feladat: } Legyen $f : \mathbb{R} \rightarrow \mathbb{R}$ egy folytonos függvény, ahol $f\left( f(x)\right) = x^{2n+1}, n \in \mathbb{N}$ és $f(1) = -1$. Igazoljuk, hogy $f$ szigorúan csökkenõ és $f(0) = 0$, valamint
$$\lim_{x \rightarrow - \infty}f(x) = - \lim_{x \rightarrow + \infty}f(x) = + \infty.$$
Adjunk példát a fenti feltételeket kielégítõ függvényekre.


\ki{Bencze Mihály}{Brassó}\medskip

{\bf 5. feladat: } Az $ABC$ derékszögû háromszögben meghúzzuk az átfogóra a magasságot. Az így keletkezett két háromszögnek megszerkesztjük a beírt köreit. Bizonyítsuk be, hogy a talppontból és ezen körök középpontjaiból alkotott háromszög hasonló az eredetihez!


\ki{Fonód Tibor}{Komárom}\medskip

{\bf 6. feladat: } Vágjunk ketté egy háromszöget egy egyenessel két egyenlõ területû részre úgy, hogy az egyenesnek a háromszögön belüli szakasza a lehetõ legrövidebb legyen.
