{\bf 1. feladat: } Bizonyítsuk be, hogy minden $k$ pozitív egész számra igaz, hogy a $(k+1)(k+2)(k+3)\cdot\dots\cdot (k+2000)$ szorzat osztható $2000^{99}$-nel! 


\ki{Oláh György}{Komárom}\medskip

{\bf 2. feladat: } Ha az $ABC$ háromszög $AB$ és $AC$ oldalán úgy vesszük fel a $D$ és $E$ pontokat, hogy $DE=DB=EC$ teljesüljön, akkor $AD=AE=BC$ is teljesül. Mekkorák az $ABC$ háromszög szögei?


\ki{Katz Sándor}{Bonyhád}\medskip

{\bf 3. feladat: } Igazoljuk, hogy bárhogyan is választunk ki a $2000$-nél nem nagyobb pozitív egész számok közül $1001$-et, biztosan lesz a kiválasztottak között két olyan szám, amelyek különbsége $4$.


\ki{Erdõs Gábor}{Nagykanizsa}\medskip

{\bf 4. feladat: } Az $ABC$ derékszögû háromszög $AB$ átfogóját a háromszögbe írt kör a $D$ pontban érinti. Bizonyítsuk be, hogy az $AD$ és $DB$ oldalhosszúságú téglalap területe egyenlõ az $ABC$ háromszög területével!


\ki{Szabó Magda}{Szabadka}\medskip

{\bf 5. feladat: } Az $1 \leq x \leq 3$ valós számokra az f függvényt a következõ módon értelmezzük:
$$f(x)=\frac{x^4-4x^3+5x^2-4x+9}{1+4x-x^2}.$$
Határozzuk meg $f$ legkisebb értékét és azt az $x$ helyet, ahol ezt a legkisebb értéket felveszi!


\ki{Árokszállási Eszter}{Paks}\medskip

{\bf 6. feladat: } Oldjuk meg az egész számok halmazán a következõ egyenletet:
$$y(1-x)^2+x(1-y)^2+(x+y)^2-x^3-y^3=2000.$$


\ki{Kovács Béla}{Szatmárnémeti}\medskip
