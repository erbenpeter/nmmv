{\bf 1. feladat: } Az $ABC$ háromszög $AB$, $BC$, $CA$ oldalain felvesszük a $D$, $E$, $F$ pontokat úgy, hogy
$${AD \over DB} = {BE \over EC} = {CF \over FA}.$$
Bizonyítsuk be, hogy a $DEF$ háromszög súlypontja egybeesik az $ABC$ háromszög súlypontjával.


\ki{Petkovics Zoltán}{Szabadka}\medskip

{\bf 2. feladat: } Ha $n>2$ egész szám, igazoljuk, hogy
$$1 + {1 \over 2} + {1 \over 3} + \ldots + {1 \over n} - \ln n \geq {1 \over 2} \left(1 + {1 \over n} + {1 \over n^2} \right).$$


\ki{Bencze Mihály}{Brassó}\medskip

{\bf 3. feladat: } Az $(a_n), n \in \mathbb{N}$ sorozatot a következõképpen értelmezzük:
$$a_1 = {1 \over 2}, \ a_2 = {1 \over 3} \, \hbox{, és} \ a_{n+2} = {a_na_{n+1} \over 3a_n - 2a_{n+1}}, \ \forall n \in \mathbb{N}^+.$$ Adjuk meg a sorozat $n$-edik tagját $n$ függvényében! 


\ki{Urbán János}{Budapest}\medskip

{\bf 4. feladat: } Az $(a_n), n \in \mathbb{N}$ sorozatot a következõ feltételeket teljesíti:
$$a_0 = a_1 = {2 \over 3} \, \hbox{, és} \, {1 \over 4}\left(1 + 2a_{n+1} +a_n^2\right) \leq a_{n+2} \leq {1 \over 3}\left(1 + a_{n+1} + a_n^2 \right).$$
Igazoljuk, hogy a sorozat konvergens és határozzuk meg a határértékét.


\ki{Dályai Pál}{Marosvásárhely}\medskip

{\bf 5. feladat: } Határozzuk meg az $f : \mathbb{R} \setminus \{-1, 1, 2\} \rightarrow \mathbb{R}$ függvényt, ha
$$f\left( {x+1 \over x-2} \right) + 2f \left( {x-2 \over x+1} \right) = x,$$
majd ábrázoljuk a függvényt grafikusan.


\ki{Balázs Lajos}{Zselíz}\medskip

{\bf 6. feladat: } Az $A, B, C$ pontok rajta vannak az $y = {1 \over x}$ egyenletû hiperbolán. Bizonyítsuk be, hogy az $ABC$ háromszög magasságpontja is ezen a hiperbolán van!


\ki{Reiman István}{Budapest}\medskip
