{\bf 1. feladat: } Adott a síkban az $A_{1}A_{2}\ldots A_{n} \left(n\geq 3\right)$ konvex sokszög, amelynél az $A_{k}A_{k+1}$ oldal hossza $a_{k}$ $\left(A_{n+1}\equiv A_{1}\right)$. Vetítsük merõlegesen a sokszöget az $A_{k}A_{k+1}$ oldalának egyenesére és jelöljük a vetület hosszát $d_{k}$-val ($k = 1, 2, \ldots, n$). Bizonyítsuk be, hogy
$$\frac{a_{1}}{d_{1}} + \frac{a_{2}}{d_{2}} + \ldots + \frac{a_{n}}{d_{n}} > 2.$$

{\bf 2. feladat: } Határozzuk meg azokat a derékszögû háromszögeket, amelyek oldalai egész számok és területének mérõszáma háromszorosa kerülete mérõszámának.


\ki{Oláh György}{Rév-Komárom}\medskip

{\bf 3. feladat: } Az $ABC$ derékszögû háromszög $AB$ átfogójához tartozó magasság talppontja $D$. Az $ADC$ és $BDC$ háromszögekbe írható körök sugara $r_{1}$ és $r_{2}$. Fejezzük ki az $ABC$ háromszögbe írható kör $r$ sugarát $r_{1}$ és $r_{2}$ segítségével.


\ki{dr. Katz Sándor}{Bonyhád}\medskip

{\bf 4. feladat: } Az $A$ halmaz a pozitív egész számokat tartalmazza $1$-tõl $2001$-ig bezárólag. Megadható-e néhány $A_{i}$ $\left(i = 1, 2, \ldots, n; n\geq 2\right)$ részhalmaz úgy, hogy teljesüljenek a következõ feltételek:
\begin{itemize}
\item{a)}$A_{1}\cup A_{2}\cup \ldots A_{n}=A$;
\item{b)}$A_{i}\cap A_{j}=\emptyset; i\neq j; i, j = 1, 2, \ldots, n$;
\item{c)}az $A_{i}$ $\left(i = 1, 2, \ldots, n\right)$
\end{itemize}
halmazban a számok közül a legnagyobb egyenlõ az összes többi szám összegével.



\ki{Balázsi Borbála}{Beregszász}\medskip

{\bf 5. feladat: } Oldjuk meg az egyenletet, ha x egész szám:
$$\frac{1}{x^{2}-4x-1} + x\left(x-1\right)^{3}=x^{3}+4x^{2}-\frac{21}{4}x.$$


\ki{Bíró Bálint}{Eger}\medskip

{\bf 6. feladat: } Az $1, 2, 3, \ldots, 2n-1, 2n$ számokat két egyenlõ csoportra osztjuk. Legyenek
$a_{1}< a_{2}< \ldots < a_{n}$ az egyik, $b_{1}>b_{2}> \ldots>b_{n}$ a másik csoport elemei nagyság szerint növekvõ, illetve csökkenõ sorrendben. Bizonyítsuk be, hogy
$$\left|a_{1}-b_{1}\right|+\left|a_{2}-b_{2}\right|+\ldots+\left|a_{n}-b_{n}\right|=n^{2}.$$
