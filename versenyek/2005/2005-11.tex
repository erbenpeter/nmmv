{\bf 1. feladat: } A valós számok $x_n$ sorozata a következõ módon van definiálva:
$$x_1=a>0,\quad x_2=b>0,\quad x_n=\frac{1+x_{n-1}}{x_{n-2}},\hbox{ ha } n\ge3.$$
Határozzuk meg az $x_{2005}$ értékét az $a$ és $b$ függvényeként!


\ki{Szabó Magda}{Szabadka}\medskip

{\bf 2. feladat: } %\newcommand\tg{\mathrm{tg}\,}

Egy háromszög oldalaira teljesül az $a^4+b^4=c^4$ összefüggés. Bizonyítsuk be, hogy ekkor a háromszög szögeire 
$\tg\alpha\cdot\tg\beta=2\sin^2\gamma$, ahol $\gamma$ a c oldallal szemközti szög.



\ki{Bogdán Zoltán}{Cegléd}\medskip

{\bf 3. feladat: } Bizonyítsuk be, hogy ha $a$, $b$ és $c$ pozitív valós számok, akkor 
$$\sqrt{\frac{a}{b+c}} + \sqrt{\frac{b}{c+a}} + \sqrt{\frac{c}{a+b}} >2.$$


\ki{Neubauer Ferenc}{Munkács}\medskip

{\bf 4. feladat: } Mennyi a tízes számrendszerben felírt $1\cdot 1! + 2\cdot 2! + 3\cdot 3! +\ldots + 2005\cdot 2005!$ szám utolsó 500 számjegyének összege?


\ki{Kacsó Ferenc}{Marosvásárhely}\medskip

{\bf 5. feladat: } Ezer birkát, melyek a 000, 001, 002, $\ldots$, 998, 999 számokkal vannak megjelölve, este a juhászkutyák száz olyan akolba terelnek be, melyek ajtajain a 00, 01, 02, $\ldots$, 98, 99 kódok olvashatók. Minden birkának olyan akolban kell éjszakáznia, melynek kódját megkapjuk, ha a birka számának valamelyik számjegyét töröljük. (Az 537-es birka tehát csak az 53, 57 vagy a 37 kódszámú akolba mehet.) Sötétedés elõtt minden birka bekerült valamelyik akolba. Mennyi az üresen maradó aklok számának lehetõ legnagyobb értéke?


\ki{Erdõs Gábor}{Nagykanizsa}\medskip

{\bf 6. feladat: } Bizonyítsuk be, hogy minden x, y valós számpár esetén fennáll a 
$$\sqrt{9 + x^2 - 3x\sqrt{3}} + \sqrt{x^2 + y^2 - xy\sqrt{3}} + \sqrt{16 + y^2 - 4y\sqrt{3}} \ge 5$$
egyenlõtlenség.
