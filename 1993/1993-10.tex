{\bf 1. feladat: } Az $ABCD$ négyzet $CD$ oldalára, a négyzet külsõ tartományában, megszerkesztjük az $M$-ben derékszögû $DCM$ háromszöget. Bizonyítsuk be, hogy a $DMC \angle$ szögfelezõje a négyzetet két egybevágó részre osztja.


\ki{Mészáros József}{Galánta}\medskip

{\bf 2. feladat: } Bizonyítsuk be, hogy a
$$93n^2 - 4n + 5$$
másodfokú polinom nem írható fel két racionális együtthatójú elsõfokú polinom négyzeteinek különbségeként.


\ki{Mészáros József}{Galánta}\medskip

{\bf 3. feladat: } Oldjuk meg a pozitív prímszámok halmazán a következõ egyenletet:
$$3x^2 + 6x = 2y^2 + 7y.$$


\ki{Oláh György}{Révkomárom}\medskip

{\bf 4. feladat: } Az $ABC$ háromszög súlyvonalai $s_a$, $s_b$, illetve $s_c$. Bizonyítsuk be, hogy a háromszög akkor és csak akkor derékszögû, ha $s_a^2 + s_b^2 = 5s_c^2$.


\ki{Neubauer Ferenc}{Munkács}\medskip

{\bf 5. feladat: } Bizonyítsuk be, hogy $n > 1$ egész szám esetén
$$\root 3 \of {n - \sqrt{n}} + \root 3 \of{n + \sqrt{n}}$$
irracionális szám.


\ki{Tar Miklós}{Ungvár}\medskip

{\bf 6. feladat: } Igazoljuk a következõ azonosságot:
$$\left[ \sqrt{n} + \sqrt{n+2} + \sqrt{n+4} \right] = \left[ \sqrt{9n+17} \right],$$
ahol $n \geq 2$ egész szám, és $[a]$ az $a$ szám egészrészét jelöli.


\ki{Bencze Mihály}{Brassó}\medskip
