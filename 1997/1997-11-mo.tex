{\bf 1. feladat: } Egy nem állandó számtani sorozat elsõ két tagjának összege és szorzata egyenlõ egymással. Az elsõ három tag összege és szorzata is egyenlõ. Határozzuk meg a sorozat elsõ négy tagjának az összegét!


\ki{Kovács Béla}{Szatmárnémeti}\medskip

{\bf 1. feladat I. megoldása: } Jelöljük a sorozat tagjait a szokásos módon! Vezessük be az $a_1 = a$ jelölést. A feltételünk szerint $a \not = a_2$, $a + a_2 = aa_2$, és $aa_2a_3 = a + a_2 + a_3$. Ezekbõl az egyenlõtlenségekbõl láthatjuk, hogy $a \not = 0$ és $a \not = 1$, valamint hogy
$$a_2 = {a \over a-1}, \ a_3 = {a^2 \over a^2 - a + 1} = {2a \over a-1}$$
A számtani sorozat tulajdonságai alapján $a + a_3 = 2a_2$, ami az elõzõekkel egybevetve azt jelenti, hogy $a^3 - 3a^2 + 3a - 1 = 2$. Ismert azonosság szerint ez $(a-1)^3 = 2$ alakba írható, amibõl $a = 1 + \root 3 \of {2}$. Így tehát a képletek alapján az elsõ négy tag:
$$a_1 = 1 + {1 + \root 3 \of {2} \over \root 3 \of {2}}, \ a_3 = {3 \over 1 + \root 3 \of {2}}, \ a_4 = 2a_3 - a_2 = {6 \over 1 + \root 3 \of {2}} - 1 - {1 \over \root 3 \of {2}}$$
Ezeket összeadva pedig az elsõ négy tag összege $1 + \root 3 \of {2} + {9 \over 1 + \root 3 \of {2}}$.
\medskip

{\bf 2. feladat: } A konvex $n$ oldalú sokszöget vágjuk szét háromszögekre. Minden háromszögbe írjunk kört. Bizonyítsuk be, hogy a körök sugarainak összege nagyobb vagy egyenlõ a ${2T \over k}$ hányadosnál, ahol $T$ az $n$ oldalú sokszög területe, $k$ pedig a kerülete!


\ki{Szabó Magda}{Szabadka}\medskip

{\bf 2. feladat I. megoldása: } Jelöljük a keletkezett háromszögek számát $m$-mel! Az $i$-edik háromszög területe legyen $t_i$, kerülete $k_i$, a beírható kör sugara $r_i$! Ismert összefüggés alapján $r_i = {2t_i \over k_i}$. Minden háromszög kerülete legfeljebb akkora lehet, mint a sokszögé, így ${2t_i \over k_i} \geq {2t_i \over k}$ bármely $i$-re. A körök sugarainak összege tehát
$$\sum_{i=1}^{m}r_i = \sum_{i = 1}^{m}{2t_i \over k_i} \geq \sum_{i = 1}^{m}{2t_i \over k} = {2T \over k},$$
és éppen ezt akartuk bizonyítani.
\medskip

{\bf 3. feladat: } Adott a síkon $n$ darab pont, amelyek között nincs három, amely egy egyenesre esne, és nincs négy, amely egy körön lenne. Minden ponthármas köré kört írunk. Mutassuk meg, hogy a körök között lévõ egységsugarú körök száma legföljebb ${n(n-1) \over 3}$.


\ki{Róka Sándor}{Nyíregyháza}\medskip

{\bf 3. feladat I. megoldása: } Bármely két adott pontra a síkon legfeljebb két egységsugarú kör illeszthetõ, mivel minden közös kör középpontja rajta van a két pont által alkotott szakasz felezõmerõlegesén, és ezen csak két megfelelõ pont lehet.

Így ha bármely két pontot kiválasztjuk, az összesen ${n \choose 2}$ pontpár, ez azt jelenti, hogy legfeljebb $2 \cdot {n \choose 2}$ egységsugarú kört rajzolhatunk meg. Ebben az esetben viszont minden kört háromszor számoltunk, tehát az egységsugarú körök száma legfeljebb
$${1 \over 3} \cdot 2 \cdot {n \choose 2} = {n(n-1) \over 3},$$
és ezt kellett belátnunk.
\medskip

{\bf 4. feladat: } Az $f(x)$ másodfokú polinomot helyettesítjük az
$$x^2 f \left( 1 + {1 \over x} \right)$$
vagy az
$$(x-1)^2 \cdot f \left({1 \over 1-x} \right)$$
polinomok közül az egyikkel. Az $x^2 + 1997x + 1998$ polinomból megkaphatjuk-e ilyen mûveletek segítségével az $x^2 + 1996x + 1997$ polinomot?


\ki{Orosz olimpiai feladat alapján Kubatov Antal}{Kaposvár}\medskip

{\bf 4. feladat I. megoldása: } Paraméterezzük meg a polinomot: $f(x) = ax^2 + bx +c$. Nézzük meg, mik lesznek a helyettesítés során kapott polinomok:
{\setlength\arraycolsep{2pt}\begin{eqnarray*}
x^2f\left(1 + {1 \over x}\right) &=& x^2\left(a\left(1 + {1 \over x}\right)^2 + b\left(1 + {1 \over x}\right) + c\right) = \\[6pt]
&=& a(x+1)^2 + bx(x+1) + cx^2 = (a+b+c+)x^2 + (2a+b)x + a \\[6pt]
(x-1)^2f\left({1 \over x-1}\right) &=& x^2\left(a\left({1 \over x-1}\right)^2 + b\left({1 \over x-1}\right) + c\right) = \\[6pt]
&=& a - b(x-1) + c(x-1)^2 = cx^2 - (2c + b)x + a+b+c \end{eqnarray*}}

Könnyen ellenõrizhetõ, hogy mindkét polinom diszkriminánsa megegyezik az eredetiével. Mivel pedig a feladatban szereplõ két polinom diszkriminánsa nem ugyanannyi, azért ilyen átalakításokkal nem kaphatjuk meg egyiket a másikból.
\medskip


\hbox to \hsize{\hskip2truecm\hrulefill\hskip2truecm}
{\bf 5. feladat: } Bizonyítsuk be, hogy érvényes: $a^2 (-u^2 + v^2 + w^2) + b^2(u^2 - v^2 + w^2) + c^2(u^2 + v^2 - w^2) \geq 16 Tt$, ahol $t$ az $a,b,c$ oldalú háromszög, $T$ pedig az $u,v,w$ oldalú háromszög területe. Mikor áll fenn az egyenlõség?


\ki{Oláh György}{Révkomárom}\medskip

{\bf 5. feladat I. megoldása: } Legyen az elsõ háromszögben a $c$ oldallal szemközti szög $\gamma$, a másikban a $w$-vel szemközti szög $\phi$. A háromszög trigonometrikus területképlete alapján $t = {ab \sin \gamma \over 2}, \ T = {uv \sin \gamma \over 2}$. Alkalmazzuk most a koszinusztételt!
$$c^2 = a^2 + b^2 - 2ab \cos \gamma; \ u^2 + v^2 - w^2 = 2uv \cos \phi$$
Ha ezeket felhasználjuk, azt kapjuk az eredeti egyenlõtlenségbõl:
$$a^2(2v^2 - 2uv \cos \phi) + b^2(2u^2 - 2uv \cos \phi) + (a^2 + b^2 - 2ab\cos \gamma)2uv \cos \phi \geq 4abuv \sin \phi $$
Ezt ekvivalensen átalakítva
$$2(a^2v^2 - b^2u^2) - 4abuv(\cos \gamma \cos \phi + \sin \gamma \sin \phi) \geq 0$$
Ezt pedig 2-vel leosztva és teljes négyzetté alakítva (felhasználva, hogy a bal oldal második tagjának második tényezõje $\cos (\gamma - \phi)$, és hozzáadva $2abuv - 2abuv = 0$-t a bal oldalhoz)
$$(av - bu)^2 + 2abuv \left(1 - \cos (\gamma - \phi)\right)$$
Mivel pedig $\cos (\gamma - \phi) \leq 1$ mindig teljesül, azért ez is igaz lesz, és mivel ekvivalens az eredeti egyenlõtlenségünkkel, azért azt is bebizonyítottuk. Egyenlõség csak akkor áll fenn, ha mindkét tag a bal oldalon 0, tehát
$$av = bu, \ \cos(\gamma - \phi) = 1$$
Ez pedig azt jelenti, hogy $\gamma = \phi$ és ${a \over b} = {u \over v}$, tehát akkor lesz egyenlõtlenség, ha a két háromszögünk hasonló.
\medskip

{\bf 6. feladat: } Igazoljuk, hogy a $|\sin n|$ alakú számok halmazának ($n$ nem negatív egész) van legalább két olyan eleme, amelyek kisebbek ${1 \over 1000}$-nél!


\ki{Dáné Károly}{Marosvásárhely}\medskip

{\bf 6. feladat I. megoldása: } A 0 eleme a halmaznak ($\sin 0 = 0$), ezért elég egyetlen további megfelelõ $n$-et találni. Mivel bármely $n,k$ pozitív egészre $|\sin n| = |\sin( n - k \pi)|$, azért elég igazolni, hogy vannak olyan $n,k$ pozitív egészek, melyeknél $|n - k\pi| < {1 \over 1000}$.

Ennek belátásához tekintsük az $\{ \pi \}, \{ 2\pi \} , \ldots , \{1001 \pi \}$ számokat. Ezek mind különbözõk lesznek, mert bármelyik kettõ egyenlõsége esetén $p \pi - [p \pi] = q \pi - [q \pi]$ állna fenn, ami viszont lehetetlen, mert ekkor $(p-q) \pi$ egész lenne, tehát $\pi$ racionális lenne. $\{ \pi \}, \{ 2\pi \} , \ldots , \{1001 \pi \}$ mindegyike 0 és 1 közé esik. Így van köztül kettõ, amelyek különbsége abszolútértékben kisebb, mint ${1 \over 1000}$. Legyenek ezek $p$ és $q$ ! Ekkor
$$\left|[p \pi] - [q \pi] - (p - q) \pi \right| < {1 \over 1000}$$
Ebben az esetben pedig $n = [p \pi] - [q \pi]$, $k = q - p$ megfelelõ értékek lesznek, $|n - k \pi| < {1 \over 1000}$.
\medskip
